# 김진욱 이력서

프론트엔드 개발자 김진욱의 이력서 페이지입니다.

🔗 **https://happy-wook-kim.github.io/resume/**

## 소개

- 단일 HTML 파일(`index.html`)로 구성된 정적 이력서 페이지입니다.
- 폰트·스타일이 모두 포함되어 있어 별도 빌드 없이 바로 렌더링됩니다.

## 배포

`main` 브랜치의 `index.html`을 GitHub Pages가 그대로 서빙합니다.
`index.html`을 수정한 뒤 `main`에 push하면 자동으로 재배포됩니다.

```bash
git clone https://github.com/happy-wook-kim/resume.git
# index.html 수정
git add index.html
git commit -m "이력서 내용 수정"
git push
```
